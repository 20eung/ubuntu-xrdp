# Ubuntu 22.04/20.04에 xRDP(Remote Desktop) 설치하기

## 목표

여기에 데스크탑 환경(GUI)을 설치하고,    
여러 운영 체제에서 원격 시스템에 연결하고,    
일부 설정을 조정하여 느린 연결을 시도하고    
속도를 높이는 방법을 배웁니다.
